# Drop real audio here (optional)

The game uses synth placeholders until it finds MP3s in this folder.
Any file you add is used automatically on next page load. Names:

- music-menu.mp3     (loops on the menu)
- music-match.mp3    (loops during matches)
- sfx-goal.mp3
- sfx-whistle.mp3
- sfx-kick.mp3
- sfx-tackle.mp3
- sfx-powerup.mp3
- sfx-morph.mp3      (field transformation)
- sfx-catch.mp3      (ball pickup)

Missing files just keep their synth fallback. Keep SFX short (<2s).
